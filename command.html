About: THE UNIX COMMAND LINE INTERFACE

" You can only call yourself a developer if you know atleast how to use the command line " Known

The Command Line Interface [CLI] is a poweful way to interact with your computer without the need for a graphical user interface [GUI].
This means, it's a way of issuing commands to the computer and getting it to do whatever it is that we want it to do through the terminal.

Important links: Git & GitHub | Hyper Terminal

Check out our github page Tech-Hub

In order to get started, we we're going to install a super customizable terminal called hyper. This is because we want to stay consistent between different operating systems ie. windows, linux and macOS.
The other reason is that hyper has got a whole bunch of plugins and themes that can be customized which makes it super efficient and poweful. Head over to hyper.is and download hyper for your system.
